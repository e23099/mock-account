## 目標
做一個模擬帳戶( python class ), 支援多幣別、多資產，就像一個模擬券商一樣。用戶交易各種金融產品，每種產品結算幣別不同，可以正確計算合理的帳戶餘額。

## 用途
- 實盤模擬：用戶於模擬交易時傳入價格、成交紀錄、參考匯率資訊，模擬帳戶維護即時損益、庫存，用戶可取得當前帳戶狀況，作為交易的依據
- 重現績效：用戶傳入歷史價格、歷史成交紀錄、歷史參考匯率，模擬帳戶重現歷史庫存、即時損益，用戶可取得歷史帳戶淨值，分析策略績效

## 需求

- 可知道各產品的庫存餘額（作多或做空 幾口/幾股）
- 可傳入產品價格、成交紀錄、參考匯率
- 時間精確，統一以 unix epoch 紀錄時間，避免不同產品交易所時區不同造成的混亂
- 可在任意時間取出當前帳戶 balance (帳戶餘額), equity (帳戶淨值), unrealized pnl (未實現損益)，給出基於當時最新價格、匯率的計算結果
- 可設定一個至多個 feeModel, 規則可包含手續費 rebate (習慣上券商會每月rebate, 當日都還是照扣)
- 可設定預設幣別，所有結果都以此幣別呈現
- 可取出歷史實現損益 （帶有實際幣別、參考幣別，方便重新用別的幣別計算）
- 可取出歷史成交明細和費用（帶有實際幣別、參考幣別，方便重新用別的幣別計算）
- 以 FIFO 原則記帳、以指定的預設幣別呈現餘額、淨值、未實現損益
